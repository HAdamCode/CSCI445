Importance of CSS

Anyone who wants to program web pages should have some understanding of CSS Some reasons paraphrased from bootcamp:

  • CSS is a fundamental part of the web
  • You (or someone you know) may want to create your own site - be an entrepeneur!
  • You can create unique websites - strut your stuff!
  • You can make more money - are you a full-stack developer?

Another reason: it will challenge how you think about programming. CSS is NOT like other languages! Embrace it and you'll expand your flexibility.

Why cats are so cool

  8. They are natural hunters
